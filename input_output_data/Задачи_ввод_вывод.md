Выходные данные
- Программа должна вывести сумму, произведение и среднее арифметическое введенных чисел. Среднее арифметическое нужно вывести с точностью 3 знака после десятичной точки.
```
Пример
Входные данные
4 5 7
Выходные данные
4+5+7=16
4*5*7=140
(4+5+7)/3)=5.333
```

*Решение:*
```

```
