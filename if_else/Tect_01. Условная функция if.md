# Условная функция if

> Синтаксис инструкции.
- Оператор-выражение if
- Оператор-выражение if-else
- Оператор-выражение if-elif-else
```
num = 11
if num > 10:
    print("Число больше 10")
```

```
num = 9
if num > 10:
    print("Число больше 10")
else
    print("Число меньше 10")
```

```
num = 10
if num > 10:
    print("Число больше 10")
elif num < 10:
    print("Число меньше 10")
else:
    print("Число равно 10")
```
